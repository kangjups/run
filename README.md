# run
## 감옥에 수감된 탈옥수의 감옥 탈출극
### 다양한 장애물 , 체력 , 스텟 
### 여러 음식들을 먹으면 점수가 오른다
### 3개의 아이템을 사용할 수 있다
### ----------------------------
### 선생은 광신도 집단 '학교'의 고위 간부이다.
### 그는 과도한 포교 활동으로 인해 경찰에 체포되었고 곧바로 교도소에 수감되었다
### 선생을 구하기 위해 학교는 케이크 안에 제트팩을 숨겨 보냈다
### -----------------------------
### 당신은 제트팩 속 AI 입이다 선생을 도와 교도소에서 탈출 하라!
### 파이게임 : 시작 화면                                                                    
### -> 시작 화면 [ 게임 시작, 설정, 나가기 ]  
### --> 설정 화면 [ 소리 ]
### --> 나가기 : 게임종료 
###                                     
### 파이 게임 : 메뉴 화면(게임 시작)                      
### -> 메뉴 화면 [ 출동, 설정, 캐릭터, 레벨, 골드, 도감 ]
### --> 설정 [ 소리 ]
### --> 캐릭터 [ 스텟(레벨), 스킬, 장비 ]   
### --> 골드 [ 상점 ] 
### --> 도감 [ 도감 ]
###
### 파이게임 : 게임화면(출동)
### -> 게임화면 [ 체력, 레벨, 점수, 캐릭터, 챕터 ]
### --> 게임종료 [ 점수, 골드 ]
